# Excel题库格式说明

## 标准格式

应用支持以下Excel格式（AI转换后的格式）：

| 列 | 标题 | 说明 | 示例 |
|----|------|------|------|
| A  | 题目类型 | 单选/多选/判断 | 单选 |
| B  | 题目内容 | 完整的题目描述 | 以下哪个是动物？ |
| C  | 选项A | 第一个选项 | 猫 |
| D  | 选项B | 第二个选项 | 桌子 |
| E  | 选项C | 第三个选项 | 书 |
| F  | 选项D | 第四个选项 | 电脑 |
| G  | 正确答案 | 正确答案 | A |

## 示例数据

```
题目类型	题目内容	选项A	选项B	选项C	选项D	正确答案
单选	以下哪个是动物？	猫	桌子	书	电脑	A
多选	以下哪些是水果？	苹果	香蕉	胡萝卜	西瓜	A,B,D
判断	太阳从东边升起	对	错			对
```

## AI提示词模板

将以下提示词发送给AI，让其帮您转换Excel题库：

```
请将以下Excel题库转换为统一格式，每行一个题目，格式如下：

题目类型|题目内容|选项A|选项B|选项C|选项D|正确答案

说明：
1. 题目类型：单选、多选或判断
2. 题目内容：完整的题目描述
3. 选项：A、B、C、D四个选项的内容（判断题型无需选项）
4. 正确答案：单选填A/B/C/D，多选填A,B,C（逗号为英文逗号），判断填对/错

示例：
单选|以下哪个是动物？|猫|桌子|书|电脑|A
多选|以下哪些是水果？|苹果|香蕉|胡萝卜|西瓜|A,B,D
判断|太阳从东边升起|对|错|||对

请严格按照此格式输出，确保分隔符为竖线|，不包含表头。
```

## 文本输入格式

如果您选择"输入文本"方式导入，请按照以下格式粘贴：

```
单选|以下哪个是动物？|猫|桌子|书|电脑|A
多选|以下哪些是水果？|苹果|香蕉|胡萝卜|西瓜|A,B,D
判断|太阳从东边升起|对|错|||对
```

注意：
- 每行一个题目
- 使用竖线 | 分隔各字段
- 判断题的选项位置留空或填对/错
- 多选题的答案用逗号分隔（如 A,B,C）
