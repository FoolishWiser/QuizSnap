# 题库助手 (QuizSnap) - Android应用

## 功能介绍

这是一个用于辅助答题的安卓应用，主要功能包括：

1. **AI提示词生成**：生成标准化的AI提示词，用于将Excel题库转换为统一格式
2. **题库导入**：支持Excel文件和文本输入两种方式导入题目
3. **题目管理**：保存题目到本地数据库，支持查看历史记录，按题库分组管理
4. **悬浮窗识别**：通过悬浮窗按钮触发OCR识别，自动匹配答案并显示
5. **自动答题（无障碍模式）**：开启无障碍服务后，可自动识别屏幕题目、自动点击答案、自动进入下一题
6. **自动答题（Shizuku 模式）**：通过 Shizuku 框架获取系统权限，无需无障碍服务即可模拟点击。支持自定义答题数量、最低总时间，每道题自动生成随机点击时间，保证总时间 ≥ 设定值 + 10 秒

## 使用流程

### 第一步：准备题库

1. 打开应用，复制主界面显示的AI提示词
2. 打开外部浏览器，将您的Excel题库和提示词一起发送给AI
3. AI会将题库转换为统一格式并返回文本结果

### 第二步：导入题库

**方式一：Excel文件导入**
1. 点击"选择Excel"按钮
2. 选择AI处理好的Excel文件（需符合指定格式）

**方式二：文本输入导入**
1. 点击"输入文本"按钮
2. 粘贴AI返回的文本结果

### 第三步：使用悬浮窗识别

1. 点击主界面的"启动悬浮窗"按钮
2. 授予悬浮窗权限（如未授予）
3. 打开需要答题的界面
4. 点击悬浮窗的"识别"按钮
5. 应用会自动识别屏幕上的题目并在悬浮窗中显示答案

### 第四步：自动答题

**方式一：无障碍模式**
1. 点击主界面的"设置"按钮
2. 开启"自动答题"开关
3. 在弹出的对话框中点击"去设置"
4. 在无障碍设置中找到"自动答题服务"并开启
5. 返回应用后即可使用

**方式二：Shizuku 模式（推荐）**
1. 在设备上安装 Shizuku 应用（酷安/官网可下载）
2. 打开 Shizuku 应用启动服务（无线调试或 root）
3. 回到本应用"设置"页面
4. 开启 "自动答题（Shizuku）" 开关
5. 点击下方"答题配置"按钮，设置：
   - 答题总数量
   - 最低总时间（秒）
   - 系统自动生成每题随机点击时间，保证总时间 ≥ 设定值 + 10 秒
6. 进入答题界面后，应用通过 Shizuku 自动完成答题

## Excel格式要求

应用期望的Excel格式（AI转换后的格式）：

| 列号 | 内容 | 说明 |
|------|------|------|
| 1    | 题目类型 | 单选/多选/判断 |
| 2    | 题目内容 | 完整的题目描述 |
| 3    | 选项A | 单选题的选项A |
| 4    | 选项B | 单选题的选项B |
| 5    | 选项C | 单选题的选项C |
| 6    | 选项D | 单选题的选项D |
| 7    | 正确答案 | A/B/C/D 或 对/错 |

## 技术特点

- **数据库**：使用Room数据库本地存储题目，支持自动响应式更新
- **OCR识别**：使用Google ML Kit进行中文文字识别
- **题目匹配**：基于编辑距离算法进行模糊匹配
- **悬浮窗**：使用Android WindowManager实现悬浮窗功能
- **ListAdapter**：使用 DiffUtil 增量更新，大数据量下流畅不卡顿
- **Shizuku**：支持 Shizuku 框架提升权限模拟点击

## 权限说明

应用需要以下权限：

1. **悬浮窗权限**：用于显示悬浮按钮和答案窗口
2. **无障碍权限**：用于自动识别屏幕内容和模拟点击（无障碍模式必需）
3. **Shizuku 权限**：Shizuku 模式用于 shell 点击操作

## 注意事项

1. OCR识别准确率受屏幕清晰度和字体影响
2. 题目匹配使用模糊匹配算法，相似度阈值默认为60%
3. Shizuku 模式需要在设备上安装 Shizuku 应用并启动服务
4. 悬浮窗功能需要手动授予权限
