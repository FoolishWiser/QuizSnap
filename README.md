# 题库助手 (QuizSnap) - Android应用

## 功能介绍

这是一个用于辅助答题的安卓应用，主要功能包括：

1. **AI提示词生成**：生成标准化的AI提示词，用于将Excel题库转换为统一格式
2. **题库导入**：支持Excel文件和文本输入两种方式导入题目
3. **题目管理**：保存题目到本地数据库，支持查看历史记录
4. **悬浮窗识别**：通过悬浮窗按钮触发OCR识别，自动匹配答案并显示
5. **自动答题功能**：开启无障碍服务后，可自动识别屏幕题目、自动点击答案、自动进入下一题

## 使用流程

### 第一步：准备题库

1. 打开应用，复制主界面显示的AI提示词
2. 打开外部浏览器，将您的Excel题库和提示词一起发送给AI
3. AI会将题库转换为统一格式并返回文本结果

### 第二步：导入题库

**方式一：Excel文件导入**
1. 点击"选择Excel"按钮
2. 选择AI处理好的Excel文件（需符合指定格式）

**方式二：文本输入导入**
1. 点击"输入文本"按钮
2. 粘贴AI返回的文本结果

### 第三步：使用悬浮窗识别

1. 点击主界面的"启动悬浮窗"按钮
2. 授予悬浮窗权限（如未授予）
3. 打开需要答题的界面
4. 点击悬浮窗的"识别"按钮
5. 应用会自动识别屏幕上的题目并在悬浮窗中显示答案

### 第四步：自动答题（可选功能）

1. 点击主界面的"设置"按钮
2. 开启"自动答题"开关
3. 在弹出的对话框中点击"去设置"
4. 在无障碍设置中找到"自动答题服务"并开启
5. 返回应用后，自动答题功能即可使用
6. 在答题界面点击悬浮窗的"识别"按钮，即可自动完成答题

## Excel格式要求

应用期望的Excel格式（AI转换后的格式）：

| 列号 | 内容 | 说明 |
|------|------|------|
| 1    | 题目类型 | 单选/多选/判断 |
| 2    | 题目内容 | 完整的题目描述 |
| 3    | 选项A | 单选题的选项A |
| 4    | 选项B | 单选题的选项B |
| 5    | 选项C | 单选题的选项C |
| 6    | 选项D | 单选题的选项D |
| 7    | 正确答案 | A/B/C/D 或 对/错 |

## 技术特点

- **数据库**：使用Room数据库本地存储题目
- **OCR识别**：使用Google ML Kit进行中文文字识别
- **题目匹配**：基于编辑距离算法进行模糊匹配
- **悬浮窗**：使用Android WindowManager实现悬浮窗功能

## 权限说明

应用需要以下权限：

1. **悬浮窗权限**：用于显示悬浮按钮和答案窗口
2. **存储权限**：用于读取Excel文件
3. **无障碍权限**：用于自动识别屏幕内容和模拟点击（自动答题功能必需）
4. **屏幕截图权限**：用于OCR识别（如需要）

## 注意事项

1. OCR识别准确率受屏幕清晰度和字体影响
2. 题目匹配使用模糊匹配算法，相似度阈值默认为60%
3. 建议题库题目数量不要过多，以免影响匹配速度
4. 悬浮窗功能需要手动授予权限
